フリートーク＆出す

全員が手札の数字を表現したらテーマに沿った会話が可能。好きなタイミングで「カードを出したいです。いいですか？」と確認し、/playで場に出します。1枚目を出した瞬間から脱出フェーズ開始。

カードは「小さい順（1→100）」で出さないと失敗。失敗したら即座に「ストップ！」して失敗したカードを表向きで場の脇へ寄せ、寄せた枚数ぶんライフを減らします。

ライフが0になるとゲームオーバー。全員の手札が0枚かつライフが残っていればステージクリア。

🏞️

カードを出す

カードの数字に応じた言葉を言ったら/playで川に出す

🏁

ゲーム終了

/endでゲームを終了

主なコマンド

コマンド 説明
/start 新しいゲームを開始
/join ゲームに参加
/theme テーマを選択
/nextstage 次のステージを準備
/draw カードを引く
/talk フリートーク開始
/play カードを出す
/nextstage ステージ2以降で使用するとライフの増減とモモちゃんの出現をチェック
/end ゲームを終了・リセット

全てのコマンドは /help で確認できます。
